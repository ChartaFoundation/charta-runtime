# Charta Runtime

This repository contains the core implementation of the Charta language runtime, including the VM, compiler, standard library, and integrations.

## Components

- **charta-core**: Core language implementation
- **charta-ir**: Intermediate Representation
- **charta-vm**: Virtual Machine
- **charta-compiler**: Compiler (textual syntax to IR)
- **charta-stdlib**: Standard library
- **charta-agents**: Agent integration
- **charta-law**: Law-as-code integration
- **charta-bindings**: Host language bindings
- **charta-editor**: Visual editor
